# engenharia-de-prompt-fundamentos
Machine Learning (ou Aprendizado de Máquina) é um subcampo da Inteligência Artificial que permite que computadores aprendam tarefas sem serem explicitamente programados para cada passo.

Em vez de um programador escrever regras rígidas do tipo "se X acontecer, faça Y", nós fornecemos grandes volumes de dados ao computador e ele usa algoritmos estatísticos para identificar padrões e tomar decisões por conta própria.
