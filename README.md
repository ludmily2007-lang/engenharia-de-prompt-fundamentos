# engenharia-de-prompt-fundamentos

Promt Genérico:
Machine Learning (ou Aprendizado de Máquina) é um subcampo da Inteligência Artificial que permite que computadores aprendam tarefas sem serem explicitamente programados para cada passo.

Em vez de um programador escrever regras rígidas do tipo "se X acontecer, faça Y", nós fornecemos grandes volumes de dados ao computador e ele usa algoritmos estatísticos para identificar padrões e tomar decisões por conta própria.

Prompt Estruturado:
Imagine que você é um técnico de futebol tentando ensinar um robô a cobrar faltas. Na programação comum, você teria que calcular cada ângulo e força do chute manualmente. No Machine Learning, você simplesmente coloca o robô para chutar a bola mil vezes: a cada chute que vai para fora, ele ajusta o movimento, e a cada gol, ele entende que aquela técnica funciona. O "aprendizado" acontece porque a máquina analisa o que deu certo e o que deu errado em todos os treinos anteriores, criando sua própria estratégia para balançar a rede.

Com o tempo, esse "atleta digital" fica tão bom que não precisa mais de instruções; ele olha para a distância da barreira e a posição do goleiro e escolhe a melhor jogada instantaneamente. Em vez de regras rígidas, o Machine Learning é como a memória muscular de um esportista: quanto mais ele pratica com dados reais, mais precisa e inteligente se torna a sua decisão no momento do jogo.
